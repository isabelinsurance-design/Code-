Los 5 Agentes del Sistema
Cada uno con objetivo, memoria y árbol de fallback
¿Qué es CrewAI?
El framework que convierte Claude en un equipo autónomo
CrewAI es el gerente del equipo de agentes. Tú defines quién es cada agente, cuál es su objetivo, qué herramientas tiene, y cómo debe actuar cuando algo sale mal. El sistema lo ejecuta solo, con memoria entre sesiones.
Sin CrewAI
Un solo Claude muy inteligente que hace todo — pero se olvida de todo cuando cierras el chat. No hay equipo. No hay memoria persistente. No hay fallback real.
Con CrewAI
5 especialistas con roles, objetivos claros y memoria. Se comunican entre ellos. El Espía le pasa el reporte al de Contenido. El Director supervisa a todos. Cuando algo falla, buscan alternativas solos.
Los 4 Conceptos Core de CrewAI
Lo que tu equipo necesita entender para empezar
🧑‍💼 Agent (Agente)
Un empleado con rol específico. Tiene: nombre, objetivo, historia de fondo y herramientas. Ejemplo: "Soy el Agente Espía. Mi objetivo es encontrar inteligencia competitiva Medicare cada semana. Uso Meta Ad Library y web search."
📋 Task (Tarea)
Lo que ese agente debe lograr. NO son instrucciones paso a paso — es un objetivo con contexto. El agente decide cómo llegar. "Entrega inteligencia competitiva. Si Meta falla, usa Google. Nunca regreses vacío."
👥 Crew (Equipo)
El grupo completo de agentes trabajando juntos. El Director asigna tareas. El Espía pasa datos al de Contenido. El de Contenido le manda el post al Publicador. Todo fluye automaticamente.
🔧 Tool (Herramienta)
Las apps que el agente puede usar. Para Isabel: Web Search (espiar), Canva MCP (diseñar), Blotato MCP (publicar), tu CRM API (leer leads), Nextiva API (disparar llamadas).
